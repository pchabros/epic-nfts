# Hardhat project with Vue client

To install node modules run:

```shell
yarn install
cd client
yarn install
```

Hardhad commands:

```shell
npx hardhat accounts
npx hardhat compile
npx hardhat clean
npx hardhat test
npx hardhat node
node scripts/sample-script.js
npx hardhat help
```

Run client:

```shell
cd client/
yarn serve
```
